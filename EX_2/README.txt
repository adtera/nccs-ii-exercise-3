Please run in the following format:
	python EX2.py time_step_length interations

For example:
	python EX2.py 0.12 200

if no arguments are provided, the code returns to default values:
	time step length = 0.01
	iterations = 300
	=> this returns results equivalent to Co = 1, as U = 1 and delta_x = 0.01
